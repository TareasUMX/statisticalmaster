Get Started Explore Features
statistical_master.py

# Load your dataset

import

statistical_master as sm

data = sm.load_dataset('clinical_trial.csv')


# Automated descriptive analysis

report = sm.descriptive_analysis(data)

report.show_histograms()

report.outliers_detection()


# Advanced regression modeling

model = sm.linear_regression(

predictors=['age', 'dose', 'baseline'],

outcome='response',

diagnostics=True

)

Comprehensive Statistical Toolkit

StatisticalMaster provides everything you need for advanced statistical analysis in a single, intuitive package.

Descriptive Analysis

Automatically calculate measures of central tendency, dispersion, shape (skewness, kurtosis), and identify outliers using Tukey's method.

  • Histograms with fitted distributions
  • Interactive boxplots
  • Normality tests

Inferential Statistics

Perform parametric and non-parametric hypothesis tests with comprehensive reporting and assumption checking.

  • t-tests, ANOVA, Wilcoxon
  • Chi-square tests
  • Confidence intervals
  • Power analysis

Advanced Modeling

Build and diagnose sophisticated statistical models with automated reporting and visualization.

  • Multiple regression
  • Logistic regression
  • ANCOVA
  • Principal Component Analysis

Interactive Visualizations

Beautiful, publication-ready graphics with interactive exploration capabilities.

Dynamic Statistical Plots

StatisticalMaster generates interactive visualizations using Plotly, allowing you to explore your data in depth. Hover over points for details, zoom into regions of interest, and customize the appearance.

Distribution Plots

Histograms with fitted probability distributions, Q-Q plots, and kernel density estimates.

Regression Diagnostics

Residual plots, leverage plots, and partial regression plots with interactive elements.

Multivariate Plots

Correlation heatmaps, PCA biplots, and cluster analysis visualizations.

Interactive Visualization Preview

Hover, zoom, and click on elements in actual implementation

Automated Data Cleaning

StatisticalMaster includes intelligent data preprocessing:

  • Outlier detection and handling
  • Missing data imputation
  • Variable transformation
  • Multicollinearity detection
data_cleaning.py

import statistical_master as sm


# Load and clean data

data = sm.load_data('raw_data.csv',

clean=True,

impute='median',

outliers='winsorize')


# Show cleaning report

data.cleaning_report()
